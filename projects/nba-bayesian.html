How NBA Games Have Evolved:
A Bayesian Approach

Rinaldi Romulus

Which team-level statistics actually predict winning in the NBA — and has that changed over time? This project applies Bayesian variable selection to compare the statistical drivers of success across two eras of basketball, before and after the modern offensive revolution.

The Question

What actually drives winning in the NBA?

Have the statistical predictors of NBA success shifted meaningfully between the pre-Curry era and the modern pace-and-space game — and can Bayesian modeling quantify that shift?

Strategic changes in basketball over the past two decades — driven by players like Stephen Curry and the rise of three-point shooting — have fundamentally altered how teams win games. Traditional metrics like assists and field goal percentage were once thought central to success. But in today's NBA, the game is played at a faster pace with higher shot volume, more transition opportunities, and a premium on defensive disruption.

This project investigates whether those strategic shifts are reflected in the data — and uses Bayesian variable selection with spike-and-slab priors to identify which statistics consistently predict game outcomes across two distinct eras.

Methodology

Spike-and-slab priors for principled variable selection

Rather than using traditional linear regression — which doesn't handle uncertainty in variable selection — this project uses Bayesian logistic regression with spike-and-slab priors. This approach quantifies uncertainty around which features matter, rather than simply reporting which are statistically significant.

01

Data Collection & Preparation

Team per-game stats sourced from NBA Advanced Stats for the 2023–24 and 2007–08 regular seasons. Removed non-predictive columns (opponent, date, point differential), converted win/loss to binary, and standardized all features using StandardScaler.

02

Spike-and-Slab Prior Construction

Each feature gets a Bernoulli inclusion indicator (p = 0.5) determining whether its coefficient is active. Non-zero coefficients are drawn from a Normal distribution with a HalfCauchy(β=1) slab standard deviation, encouraging sparsity while allowing flexibility.

03

Bayesian Logistic Regression

The model is defined as logits = intercept + Xβ where β is modulated by the inclusion indicator. Likelihood is modeled as Bernoulli(logit_p = logits), allowing the model to learn which features genuinely predict winning.

04

MCMC Posterior Sampling

Posterior distributions sampled using Markov Chain Monte Carlo (MCMC) via PyMC. Inclusion probabilities extracted for each feature — a probability near 1.0 means the variable is a strong, consistent predictor of game outcomes.

Conclusion

The game has changed — and the data proves it

The Bayesian analysis confirms a meaningful statistical shift in what drives NBA success. In 2023–24, a small cluster of variables — personal fouls, turnovers, shot attempts, blocks, and steals — emerged as dominant predictors with inclusion probabilities of 1.00. This points to a game defined by pace, possession control, and defensive disruption.

In contrast, the 2007–08 season showed a near-uniform distribution of inclusion probabilities around 0.50, meaning no single statistic stood out as a reliable predictor. This reflects a more balanced, less specialized era of basketball — before the three-point revolution redefined offensive strategy.

By applying spike-and-slab priors rather than traditional regression, this project goes beyond identifying which variables correlate with winning — it quantifies the certainty of that relationship across different eras, offering a principled framework for understanding how the game has evolved.
